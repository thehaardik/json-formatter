# 📋 JSON Formatter

Format, validate, and minify JSON with syntax highlighting and tree view. Paste your JSON, get clean formatted output — with error detection.

## 🚀 Features

- **Instant Formatting** — Paste JSON, see formatted output
- **Syntax Highlighting** — Color-coded JSON structure
- **Tree View** — Collapsible JSON tree visualization
- **Minify** — Compress JSON to single line
- **Validate** — Real-time error detection with line numbers
- **Copy & Download** — One-click copy or download as file
- **No Server** — Runs 100% in browser

## 🎯 Why This?

Most JSON formatters are online tools with ads. This one is clean, fast, and private.

## 🛠 Tech Stack

- Next.js 14
- React
- Tailwind CSS
- shadcn/ui components

## 📦 Setup

```bash
git clone https://github.com/thehaardik/json-formatter.git
cd json-formatter
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

| Feature | Description |
